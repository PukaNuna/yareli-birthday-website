# 🎂 Interactive Birthday Website

A personalized interactive birthday website I built using HTML, CSS, and JavaScript as a surprise gift.

## ✨ Features

- Animated loading screen
- Interactive welcome experience
- Floating navigation button
- Personalized love letter
- Embedded YouTube song
- Interactive gift reveals
- Memory gallery
- Custom CSS illustrations of two cats
- Responsive design
- Published with GitHub Pages

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## 💡 What I Learned

This project helped me practice:

- HTML page structure
- CSS animations and responsive layouts
- JavaScript DOM manipulation
- Event listeners
- GitHub repositories
- GitHub Pages deployment
